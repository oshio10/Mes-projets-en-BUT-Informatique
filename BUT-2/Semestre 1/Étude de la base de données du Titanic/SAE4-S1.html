Utiliser Postgresql distant (sur le cloud ElephantSQL)

1- Créer un compte sur elephantsql.com

2- Créer une instance de base de données

3- Dans les détails de l'instance (menu à gauche), vous avez l'ensemble des détails pour se connecter à distance de votre PC

4- Lancer votre notebook local, puis voici l'exemple

// à modifier selon vos informations

DB_HOST="isilo.db.elephantsql.com"

DB_NAME="dehyldje"

DB_USER="dehyldje"

DBB_PASS="LE VOTRE"

pg_conn = psycopg2.connect(dbname=DB_NAME, user=DB_USER, password=DBB_PASS, host=DB_HOST)

cur = pg_conn.cursor()

cur.execute('''CREATE TABLE mytable(foo varchar(5), bar integer);''')

cur.execute('''INSERT INTO mytable VALUES ('bar', 4);''')

pg_conn.commit()

pg_conn.close()
